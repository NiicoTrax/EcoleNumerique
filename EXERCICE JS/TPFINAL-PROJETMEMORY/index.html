Règle du Jeu

Le but du jeu de Memory est de trouver toutes les paires de cartes identiques. Chaque fois que vous cliquez sur deux cartes, elles se retournent. Si les cartes sont identiques, elles restent retournées. Sinon, elles se retournent de nouveau. Le jeu se termine lorsque toutes les paires sont trouvées.
